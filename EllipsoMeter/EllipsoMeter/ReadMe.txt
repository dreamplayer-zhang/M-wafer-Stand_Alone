================================================================================
    MFC 라이브러리 : EllipsoMeter 프로젝트 개요
================================================================================

응용 프로그램 마법사에서 이 EllipsoMeter 응용 프로그램을 
만들었습니다. 이 응용 프로그램은 MFC의 기본 사용법을 보여 줄 뿐만 아니라 응용 
프로그램작성을 위한 기본 구조를 제공합니다.

이 파일에는 EllipsoMeter 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.

EllipsoMeter.vcxproj
    응용 프로그램 마법사를 사용하여 생성된 VC++ 프로젝트의 주 프로젝트 파일입니다. 
    파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한 
    플랫폼, 구성 및 프로젝트 기능에 대한 정보가 들어 있습니다.

EllipsoMeter.vcxproj.filters
    응용 프로그램 마법사를 사용하여 생성된 VC++ 프로젝트의 필터 파일입니다. 
    이 파일에는 프로젝트의 파일과 필터 간의 연결 정보가 들어 있습니다. 이러한 
    연결은 특정 노드에서 유사한 확장명으로 그룹화된 파일을 표시하기 위해 
    IDE에서 사용됩니다. 예를 들어 ".cpp" 파일은 "소스 파일" 필터와 연결되어 
    있습니다.


EllipsoMeter.h
    응용 프로그램의 기본 헤더 파일입니다. 여기에는 다른 프로젝트 관련
    헤더(Resource.h 포함)가 들어 있고 CEllipsoMeterApp 응용 프로그램
    클래스를 선언합니다.

EllipsoMeter.cpp
    응용 프로그램 클래스 CEllipsoMeterApp이(가) 들어 있는 기본 응용 프로그램
    소스 파일입니다.

EllipsoMeter.rc
    프로그램에서 사용하는 모든 Microsoft Windows 리소스의 목록입니다.
 여기에는 RES 하위 디렉터리에 저장된 아이콘, 비트맵 및 커서가
    포함됩니다. 이 파일은 Microsoft Visual C++에서 직접
    편집할 수 있습니다. 프로젝트 리소스는 1042에 있습니다.

res\EllipsoMeter.ico
    응용 프로그램의 아이콘으로 사용되는 아이콘 파일입니다. 이 아이콘은
    주 리소스 파일인 EllipsoMeter.rc에 의해 포함됩니다.

res\EllipsoMeter.rc2
    이 파일에는 Microsoft Visual C++ 이외의 다른 도구에서 편집한 리소스가 
    들어 있습니다. 리소스 편집기로 편집할 수 없는 모든 리소스는
    이 파일에 넣어야 합니다.

EllipsoMeter.reg
    프레임워크에서 자동으로 설정되는 등록 설정의 종류를 보여 주는 예제
    .reg 파일입니다. 이 파일을 사용자 응용 프로그램의
    .reg 파일로 사용할 수 있습니다.

EllipsoMeter.idl
    응용 프로그램의 형식 라이브러리에 대한 인터페이스 정의 언어 소스 코드가
    들어 있습니다.


/////////////////////////////////////////////////////////////////////////////

응용 프로그램 마법사에서 하나의 대화 상자와 자동화 프록시 클래스가 만들어집니다.

EllipsoMeterDlg.h, EllipsoMeterDlg.cpp - 대화 상자
    이 파일에는 CEllipsoMeterDlg 클래스가 들어 있습니다. 이 클래스는
    응용 프로그램의 주 대화 상자 동작을 정의합니다. 대화 상자의 템플릿은
    Microsoft Visual C++에서 편집할 수 있는 EllipsoMeter.rc에 있습니다.

DlgProxy.h, DlgProxy.cpp - 자동화 개체
    이 파일에는 CEllipsoMeterDlgAutoProxy 클래스가 들어 있습니다. 이 클래스는
    자동화 컨트롤러가 대화 상자에 액세스하기 위해 사용할 수 있는 자동화 메서드 및 속성의
    노출을 관리하므로 대화 상자에 대한 자동화 프록시 클래스라고 합니다. 모달 대화 상자
    기반 MFC 응용 프로그램의 경우 자동화 개체를 사용자 인터페이스와 별도로
    유지하는 것이 보다 명확하고 편리하므로 이러한 메서드 및 속성은 대화 상자
    클래스에서 직접 노출되지 않습니다.

기타 기능:

ActiveX 컨트롤
    응용 프로그램에서 ActiveX 컨트롤을 사용할 수 있도록 지원합니다.

Windows 소켓
    응용 프로그램에서 TCP/IP 네트워크를 사용한 통신을 지원합니다.

기타 표준 파일:

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더 파일(PCH)인 EllipsoMeter.pch와
    미리 컴파일된 형식 파일인 StdAfx.obj를 빌드하는 데 사용됩니다.

Resource.h
    새 리소스 ID를 정의하는 표준 헤더 파일입니다.
    Microsoft Visual C++에서 이 파일을 읽고 업데이트합니다.

기타 참고:

응용 프로그램 마법사는 "TODO:"를 사용하여 추가하거나 사용자 지정해야 하는
소스 코드 부분을 나타냅니다.

응용 프로그램이 공유 DLL에서 MFC를 사용하는 경우 해당 MFC DLL을 
재배포해야 합니다. 그리고 응용 프로그램이 운영 체제의 로캘과 
다른 언어를 사용하는 경우 해당 지역화된 리소스인 MFC100XXX.DLL을 
재배포해야 합니다. 이 두가지 항목에 대한 자세한 내용은 
MSDN 설명서의 Visual C++ 응용 프로그램 재배포 항목을 
참조하십시오.
